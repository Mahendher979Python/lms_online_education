# lms_learning
LMS Learning Platform is a Django-based web application designed for online education and training. The system provides course management, student registration, video learning, quizzes, assignments, certificate generation, user authentication, progress tracking, and an admin panel for managing educational content efficiently.
# LMS Learning Platform

A comprehensive Learning Management System (LMS) developed using Django and Python.

## Features
- User Registration & Login
- Student Dashboard
- Instructor Dashboard
- Course Management
- Video Lessons
- Quizzes & Exams
- Assignments Submission
- Progress Tracking
- Certificate Generation
- Notifications
- Admin Panel
- Responsive Design

## Technology Stack
- Python
- Django
- HTML5
- CSS3
- JavaScript
- Bootstrap
- SQLite / PostgreSQL
- Git & GitHub
